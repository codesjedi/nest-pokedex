# Pokedex
## Development

1. Clone the repo
2. Run
   ```
    yarn install
   ```
3. Install @nestjs/cli
   ```
    npm i -g @nestjs/cli
   ```
4. Run the database
   ```
    docker-compose up -d
   ```

### Tech Stack
1. Nestjs
2. Typescript
3. Mongo
